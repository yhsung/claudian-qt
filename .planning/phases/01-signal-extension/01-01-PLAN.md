**1. src/claudeprocess.h** -- Add `#include <QJsonObject>` after line 3 (`#include <QProcess>`), then add one signal declaration after line 23 (`void errorOccurred(const QString &msg);`):

```cpp
void resultReceived(const QJsonObject &result);
```

The `QJsonObject` forward declaration is not sufficient for signal parameters -- the full include is required for MOC to generate the correct marshalling code.

**2. src/claudeprocess.cpp** -- Replace lines 101-103 (the `result` branch in `parseLine()`) with a nested if/else that preserves the error path (per D-02) and emits the raw QJsonObject on success (per D-01):

BEFORE:
```cpp
} else if (type == "result" && obj["is_error"].toBool()) {
    emit errorOccurred(obj["result"].toString());
}
```

AFTER:
```cpp
} else if (type == "result") {
    if (obj["is_error"].toBool()) {
        emit errorOccurred(obj["result"].toString());
    } else {
        emit resultReceived(obj);
    }
}
```

This is the complete change. No other files are modified. No new member variables. No ClaudeBridge wiring (that is Phase 2).
